# Story SDK-5.5: Testes de Integração

**Story:** SDK-5.5
**Prioridade:** Alta (P0)

Criar testes de integração com `@SpringBootTest`.

2. **WireMock:** Usar para mockar Auth Server
3. **End-to-End:** Testar fluxo completo
